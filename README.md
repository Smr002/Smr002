## 🧑‍💻 About Me

I'm a **Software Engineer at Administrative Claim Services** (Boston, MA) — building secure, scalable backend systems for insurance claims processing: EDI/X12 pipelines, SFTP encryption, real-time WebSocket architectures, and AI inference in production.

Outside of work, I ship creative web experiences — 3D cinematic sites, luxury e-commerce, tour platforms, and multi-language chat applications — mostly for clients and businesses I care about.

- 🌍 Based in **Tirana, Albania** · Working remotely for US clients
- 🔐 Security-first mindset — pen testing, OWASP, hardened PHP/Node stacks
- 💬 Built a **real-time multi-language chat app** (WebSockets, NLP translation, multi-locale support)
- 🎨 Side passion: **immersive 3D web** (Three.js, TresJS, GSAP, Blender)
- 🤖 AI experience: model serving, deep learning (TensorFlow, Keras), inference APIs
- 📫 Reach me: [LinkedIn](https://www.linkedin.com/in/kristi-samara-a6a408294) · [Portfolio](https://web-wizz.vercel.app/)
